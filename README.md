check_rds_events
========================

Nagios check for important AWS RDS events

```
usage: check_rds_events [-h] -d DBNAME [-r REGION]

optional arguments:
  -h, --help            show this help message and exit
  -d DBNAME, --dbname DBNAME
                        RDS DB name
  -r REGION, --region REGION
                        AWS Region - default is us-west-1
```
